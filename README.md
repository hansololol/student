# 학생 성적 계산기 프로젝트

학생 과목 점수의 합, 평균, 등수를 계산하는 프로그램을 제작하였습니다<br>
학생 등수를 입력시 trigger 기능을 사용하여 자동으로 합계를 계산하여 등수가 출력될 수 있도록 설계하였습니다.
<br>

## 개발 환경 

1. 프레임워크 : spiringboot4
2. 사용언어 : JAVA, JQUERY, JSP, JAVASCRIPT, SQL, HTML, CSS
